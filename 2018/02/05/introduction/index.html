2018-02-05

Github + Hexo 搭建一个自己的博客

有人说，美国人常常会觉得奇怪，为什么中国人可以非常清楚地说出我们是由56个民族组成的国家？他们好像搞不清楚自己到底有多少个民族，因为美国不但有不同肤色的人，而且就算是同一种肤色的人，其成分也非常复杂。种族问题一直困扰着美国人，一次又一次地冲击着美国社会的基本信念和主流价值观。

美国北方人对于废除奴隶制的迫切甚至极端的要求，与其移民史有密切关联。而在美国南方，像南卡罗来纳和佐治亚这样的顽固蓄奴州，也因文化传统与经济利益而有自己的信念。一两百年来，美国人为自身的信仰不断发生激烈斗争。
1955年12月1日，黑人妇女罗莎·帕克斯因为在蒙哥马利市的公交车上拒绝给白人让座而被判入狱两年，黑人广泛抗议这明显的侮辱。马丁•路德•金领导抗议组织拉开了美国黑人民权运动序幕，最后律师宣布公共汽车公司无权在车上把黑人和白人分开。

1963年马丁•路德•金与肯尼迪总统见面，要求通过新的民权法，给黑人以平等的权利，8月28日，抗议组织在华盛顿特区组织了一次二十五万人的集会，争取种族平等。马丁·路德·金在林肯纪念馆的台阶上发表了著名演讲《我有一个梦想》，标志着20世纪黑人民权运动进入高潮。演讲结束后各大报刊纷纷转载，激励着人们为争取自由平等而不懈抗争，金的话被刻在了纪念堂的台阶上。

2013年8月28日，美国首位黑人总统奥巴马也站在华盛顿林肯纪念堂前的台阶上发表讲话，他谈了“为工作和自由向华盛顿进军”大游行过去半个世纪以来美国发生的变化，以此纪念《我有一个梦想》发表50周年。

世人都知道《我有一个梦想》以诗一般优美的语言、感人的情感和信念而著称。

但是许多人忽略了《我有一个梦想》发表时，美国国内对黑人和白人的态度其实是有很大分歧的。

黑人自然是感觉到社会中的不公正待遇，但是反对者也有很多证据证明黑人的受教育程度低，黑人聚居的地方更落后更缺乏安全感。

所以，当我们再次谈起这篇著名演讲的时候，一定要注意到它取得成功不仅仅是因为它语言激越、生动，更是因为它的价值观——对自由的信仰。

当然，《我有一个梦想》使用大量比喻、排比，推波助澜，增强了语言如大河奔流一般的气势。但是，如果只有这种感情的鼓动，而没有实实在在的道理让人反省的话，公众早晚是会清醒的。

马丁·路德·金以及追随他的人们，在相当长一段时间里，并不代表多数，而只是少数，他所面对的对手可以说并不是美国联邦政府，而是南方白人，这些人在市议会中占据支配地位，甚至是权力机构的执掌者。这些人长时间代表着另一种“民意”，这种“民意”不仅是地方政府要优先考虑的，中央政府也不敢小窥。另外，马丁·路德·金及其追随者所承受的巨大压力还来自另一种“多数”──麻木、冷淡的多数人和一些作出爆炸、刺杀、恐吓激烈行动的人。

努力争取更多数人的支持，就是马丁·路德·金演讲要追求的效果。

若是只强调公平的不可得，那自然很难得到利益既得者的支持。只有从“自由”的高度出发，一步步深化理想，才能争取更多支持。毕竟，对于自由、平等的追求是人类千百年来共同的理想，是人的哲学之源。

对这一点，马丁·路德·金毫不含糊，他在演讲一开始就说：

100年前，一位伟大的美国人，即美国第16位总统亚伯拉罕·林肯签署了《解放黑奴宣言》……然而100年后，黑人依然没有得到自由，黑人依然在种族隔离的镣铐和种族歧视的枷锁下生活……黑人依然在美国社会中向隅而泣，依然感到自己是故土家园中的流亡者。

金用聪明的修辞策略强化情绪落差，同时用上了“不能自己打脸”的说服策略：

我们共和国的缔造者草拟宪法和独立宣言的气壮山河的词句时，曾向每一个美国人许下了诺言，他们承诺所有人——不论白人还是黑人——都享有不可让渡的生存权、自由权和追求幸福权。

就有色公民而论，美国显然没有实践她的诺言。美国没有履行这项神圣的义务，只是给黑人开了一张空头支票，支票上盖着“资金不足”的戳子后便退了回来。但是我们不相信正义的银行已经破产，我们不相信，在这个国家巨大的机会之库里已没有足够的储备。因此今天我们要求将支票兑现–这张支票将给予我们宝贵的自由和正义保障。

金在演讲后半部份说：

在争取合法地位的过程中，我们不要采取错误的做法。不要为了满足对自由的渴望而抱着敌对和仇恨之杯痛饮。我们斗争时必须永远举止得体，纪律严明。我们不能容许我们的具有崭新内容的抗议蜕变为暴力行动。我们要不断地升华到以精神力量对付物质力量的崇高境界中去。

这表现出他的卓越判断力、自我克制力和妥协精神。他明白抗议组织得以发展和壮大还需两个重要的条件：一个是立宪体制和法治，这意味着对手接受法律，遵循可以预知的游戏规则，群众能变得理性节制也有赖于健全的法治。另一个条件就是强大而自由的、遵循真实和道德真理的新闻媒体。

他还提醒：

现在黑人社会充满着了不起的新的战斗精神，但是不能因此而不信任所有的白人。因为我们的许多白人兄弟已经认识到，他们的命运与我们的命运是紧密相连的，他们今天参加游行集会就是明证。

若没有这些思想的基础，《我有一个梦想》里的那些漂亮的修辞，就不能长久地发挥那么强大的鼓动作用
